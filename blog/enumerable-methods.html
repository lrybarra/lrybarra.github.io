Enumerable#cycle


For a moment, imagine that you are in the Tron universe, watching a light cycle battle. The cycles seem to have infinite power and will move until something interrupts and stops them.

This is how you can think of the Enumerable cycle method. If you call it but do not pass it any arguments, it will continue forever until it is somehow "crashed" (pressing Ctrl + C).

In a similar vein, cycle is a very powerful method that on the surface seems very simple but it can have very complex applications, such as initializing a deck of playing cards.

When you want to use cycle, you'll want to pass it a code block so that it can actually accomplish something meaningful. Don't forget to pass it an integer argument so that it knows its limit.

Here is a simple example of how you can use cycle

chant = ["Hip", "Hip", "Hooray"]

chant.cycle(3) { |m| puts m }

=begin
result:
Hip
Hip
Hooray
Hip
Hip
Hooray
Hip
Hip
Hooray
=end

 Try to think of some other useful things that cycle can do. By doing this, you will expand your programming knowledge and add another method to your toolbox.
