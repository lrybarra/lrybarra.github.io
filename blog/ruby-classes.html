Ruby Classes


In order to visualize a Ruby class we will model a component stereo system
The Stereo class will have attributes make, model, maximum wattage output, and power (on / off).
Our first method, toggle_power will switch the power on and off.
We have methods to increase and decrease volume.

class Stereo

attr_accessor :make, :model, :max_watts, :power def initialize(make, model, wattage, power=false)
@make = make
@model = model
@max_watts = wattage
@power = false

@volume = 0
end

def toggle_power
@power == true ? @power = false : @power = true
end

def increase_volume
@volume += 1
end

def decrease_volume
@volume -= 1
end

end
